# 🚀 WhatSub — 구독 관리의 모든 것

> **WhatSub(왓썹)** 은 흩어져 있는 나의 모든 구독 서비스를 한 곳에서 관리하고,  
> 다음 결제일을 놓치지 않도록 도와주는 **개인 구독 관리 서비스**입니다.

<br/>

## 📌 주요 기능

### 🔔 1. 곧 결제될 구독 자동 표시
- 다음 결제일이 가까운 구독을 홈 화면에서 자동으로 정렬해 표시
- 오늘 기준으로 **남은 일수**를 직관적으로 확인 가능

### 📋 2. 전체 구독 목록 관리
- 모든 구독을 한 번에 확인할 수 있는 목록 페이지
- 카테고리, 결제 주기, 가격 등 상세 정보 표시
- 구독 수정 및 삭제 기능 제공

### ➕ 3. 새로운 구독 추가
- 서비스명, 카테고리, 가격, 결제 주기 입력
- 결제일 입력 시 다음 결제일 자동 계산

### 📈 4. 월 지출 & 연 지출 자동 계산
- 매달, 매년 구독 소비 금액을 자동 산출
- 지출 현황을 직관적인 UI로 시각화

### 👥 5. N빵 공유 기능 (옵션)
- 구독을 몇 명이서 나누는지 입력하면  
  → 1인당 실제 부담 금액 자동 계산
